# Dashboard de Receitas Municipais - Jaguapitã

Este repositório contém um dashboard interativo para análise das receitas do município de Jaguapitã, abrangendo o período de 2021 a 2025.

## Conteúdo

- Dashboard interativo com visualizações de dados das receitas municipais
- Relatório técnico completo em PDF
- Possibilidade de exportar os dados em formato CSV

## Funcionalidades

O dashboard permite analisar:

1. **Visão Geral**: Evolução das receitas totais e composição percentual das receitas
2. **Receitas Tributárias**: Análise detalhada da evolução das receitas tributárias
3. **Impostos**: Comparativo dos principais impostos municipais (IPTU, ISS, ITBI, IRRF)
4. **Análise de Crescimento**: Variação percentual anual por categoria de receita
5. **Receita Corrente Líquida (RCL)**: Evolução da RCL ao longo do período

## Como utilizar

1. Acesse o dashboard online através do GitHub Pages: [link]
2. Para ver o código fonte, explore os arquivos deste repositório
3. Para baixar o relatório técnico completo, clique no botão "Baixar Relatório Completo (PDF)" no topo da página
4. Para exportar os dados para análise própria, clique no botão "Exportar dados (CSV)"

## Tecnologias utilizadas

- HTML/CSS
- JavaScript
- React
- Recharts (biblioteca de visualização de dados)

## Instalação local

Se desejar executar o dashboard localmente:

1. Clone este repositório
```bash
git clone https://github.com/seu-usuario/dashboard-receitas-jaguapita.git
```

2. Navegue até a pasta do projeto
```bash
cd dashboard-receitas-jaguapita
```

3. Abra o arquivo `index.html` em um navegador web

## Sobre os dados

Os dados apresentados neste dashboard são provenientes dos relatórios contábeis oficiais do município de Jaguapitã para o período de 2021 a 2025. A análise enfoca especialmente o ano de 2025, servindo como instrumento para que a nova gestão municipal compreenda o comportamento das receitas e tome decisões embasadas.
